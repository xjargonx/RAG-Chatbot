**How to run the app**

1.(RECOMMENDED) Follow the public link of the fully deployed RAG Agent: 
https://rag-chatbot-kuaaxrwrv2c6ktfzynrsmd.streamlit.app/

2. double-click the run_app.bat file, requires you to have own google gemini api key. Enter your key in secrets file in the .streamlit folder.

3. Manually run the app.py file, requires you to have your own google gemini api key. Enter your key in secrets file in the .streamlit folder.
